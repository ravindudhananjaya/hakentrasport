# Haken Transport - Employee Transport Management System

## 📱 Overview

**Haken Transport** is a comprehensive mobile application designed to streamline employee transportation management for companies. Built with Flutter and Firebase, this app provides an efficient solution for managing passenger attendance, scheduling, and transport logistics.

## ✨ Key Features

### 🚌 Driver Mode
- **Passenger Check-In**: Mark passengers as picked up, absent, or self-traveling
- **Real-time Attendance Tracking**: Track passenger status with visual indicators
- **Health Monitoring**: Record passenger health conditions and body temperature
- **Smart Scheduling**: View passengers organized by time slots and shifts
- **Contact Integration**: Quick-dial passenger phone numbers directly from the app
- **PDF Export**: Generate and print monthly attendance reports

### 🛡️ Admin Mode
- **Employee Management**: Add, edit, and manage passenger profiles
- **Route Planning**: Organize passengers by pickup locations and companies
- **Weekly Scheduling**: Manage transport schedules across 5 weeks per month
- **Statistics Dashboard**: View attendance trends and transport analytics
- **Multi-Company Support**: Handle passengers from different companies with color-coded organization
